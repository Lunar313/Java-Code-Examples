# Java-Code-Examples

A collection of Java code examples to practice and improve coding skills. Feel free to contribute, correct, and suggest improvements!

## 📁 Project Structure

```
src/
└── examples/
    ├── HelloWorld.java    - Basic Java program structure
    ├── Calculator.java    - Simple calculator with input validation
    └── Person.java        - OOP basics with encapsulation
```

## 🚀 How to Run

```bash
# Navigate to the src directory
cd src

# Compile all examples
javac examples/*.java

# Run any example
java examples.HelloWorld
java examples.Calculator
java examples.Person
```

## 💡 Java Coding Tips

### 1. **Naming Conventions**
- Classes: `PascalCase` (e.g., `HelloWorld`, `Calculator`)
- Methods & Variables: `camelCase` (e.g., `getName`, `userAge`)
- Constants: `UPPER_SNAKE_CASE` (e.g., `MAX_VALUE`)
- Packages: lowercase (e.g., `examples`, `com.myapp`)

### 2. **Code Organization**
- One class per file
- Keep methods short and focused (Single Responsibility)
- Use meaningful names that describe purpose

### 3. **Best Practices**
- Always validate user input
- Use encapsulation (private fields, public getters/setters)
- Handle exceptions appropriately
- Add Javadoc comments for public methods
- Override `toString()` for better debugging

### 4. **Common Mistakes to Avoid**
- Don't ignore exceptions (catch and handle them)
- Avoid magic numbers (use named constants)
- Don't make everything public
- Remember to close resources (use try-with-resources)
